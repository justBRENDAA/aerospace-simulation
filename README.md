# aerospace-simulation
C++ aerospace simulation modeling 2D spacecraft dynamics using time-step integration and physics-based control systems.

## Overview
This project simulates spacecraft motion in a 2D environment by updating system state variables (position, velocity, acceleration) over fixed time intervals. User controlled thrust inputs influence acceleration, which affects velocity and position over time.

The purpose of this project is to build foundational understanding of:
- Time-step simulation
- Basic physics-based motion
- Control input handling
- System state modeling in C++

## Features (Planned)
- Thruster-based acceleration
- Gravity modeling
- Velocity and position updates
- Crash detection
- Fuel consumption modeling
- Telemetry-style console output

## Technologies
- C++
- Object-Oriented Programming
- Discrete simulation logic
